Aqui serão utiizadas tags aprendidas ate o momento e criação de listas, como no exemplo a baixo:

Tipos de Lista:

Lista ordenada "ol" ; Uma lista que é colocada em ordem automaticamente. Como em uma lista de compras.

    Lista de compras 1

  1. Arroz
  2. Carne
  3. Salada
  4. Feijão

Lista "ul" ; Uma lista que não possui ordem, os itens da lista são apenas mostrados em tópicos.

    Lista de compras 2

  • Arroz
  • Carne
  • Salada
  • Feijão
